# 《窑言》叙事规范

> 本规范旨在确保游戏叙事**紧扣核心主题、保持湖湘底色、强化玩家体验**，避免哲学说教、文化堆砌或节奏失控。
>  所有内容创作（文本、台词、谜题、UI）须遵守以下原则。

------

## 一、核心原则

### 1. **主题先行：一切服务于“自我是寻找还是塑造？”**

- 每句台词、每个物品、每种气味，必须能回答：
  **“它如何帮助玩家思考‘我’的来源与意义？”**
- 禁止添加与主题无关的支线、彩蛋或背景故事。

### 2. **展示，而非讲述（Show, Don’t Tell）**

- 不直接解释哲学概念（如“存在主义”“荣格阴影”）；

- 用

  行为、环境、机制

  传递思想：

  - 想表达“记忆不可靠”？→ 让两个NPC对同一事件给出矛盾叙述；
  - 想表达“自我可塑”？→ 让玩家外形影响结局形态。

### 3. **湖湘元素必须可感、可用、可玩**

- 文化符号需转化为**交互元素**（如湘剧旋律=解谜节奏，铜官窑=气味来源）；
- 禁止纯装饰性引用（如墙上挂“实事求是”匾但无功能）。

------

## 二、文本与台词规范

### 1. **主角无台词**

- 玩家通过**行动、选择、备注**表达意志；
- 任何“内心独白”均以**环境反馈**呈现（如显形时幽灵闪烁）。

### 2. **NPC台词要求**

| 要求         | 说明                           | 示例                                 |
| ------------ | ------------------------------ | ------------------------------------ |
| **简短**     | 单句 ≤ 15 字，避免长段落       | “扫干净，心才安。”                   |
| **具象**     | 用具体物象表达情感             | “火好大……手好烫。”（不说“我很悲伤”） |
| **方言点睛** | 仅用1–2个长沙方言词/句，不滥用 | “莫乱跑”“心才安”                     |
| **留白**     | 说完即停，不解释               | “你是谁不要的？”（说完转身）         |

### 3. **关键文本风格**

- **字幕/UI文字**：仿古线装书手写体，墨色随情绪变化（执念→黑，释然→淡）；
- **备注本内容**：玩家输入限10字内，系统自动补全为文言短句（如输入“校工怕黑” → 显示“守院者畏暗”）。

------

## 三、叙事节奏控制

### 1. **三章节奏模型**

| 章节             | 情绪曲线           | 节奏策略                       |
| ---------------- | ------------------ | ------------------------------ |
| **第一章：问器** | 迷茫 → 好奇 → 犹豫 | 缓慢引入机制，给予充分探索时间 |
| **第二章：观人** | 共情 → 怀疑 → 震动 | 谜题稍难，制造认知冲突         |
| **第三章：成我** | 震惊 → 沉思 → 决断 | 减少操作，强化沉浸与选择重量   |

### 2. **防卡关设计**

- 若玩家5分钟未推进，通过**环境提示**引导（如油灯微闪、风声指向）；
- 所有谜题提供**双重线索**（视觉 + 听觉 / 气味 + 文本）。

------

## 四、多结局叙事规则

### 1. **结局由行为决定，非数值隐藏**

- 玩家应能清晰感知自己倾向哪类结局：
  - 多次显形助人 → 知道自己走向【塑我】；
  - 拒绝命名、隐身离开 → 知道自己走向【寻我】。

### 2. **结局文本极简**

- 每结局仅1句字幕 + 1个意象画面；
- 禁止解释、总结或道德评判。

### 3. **轮回暗示克制**

- 仅结尾显示“这是第 __ 次苏醒。”
- 数字 = 玩家收集的气味总数（最小3，最大9），**不透露总次数上限**。

------

## 五、禁忌清单（严禁出现）

❌ 直接提及历史事件名称（如“文夕大火”“抗战”）——只通过物品日期、气味折射。
 ❌ 主角自述身份或哲学观点（如“我是影子”“存在先于本质”）。
 ❌ NPC长篇回忆或背景故事（每人仅1段核心记忆）。
 ❌ 湖湘名言堆砌（如连续引用曾国藩、毛泽东语录）。
 ❌ 多余UI文字（如“按E穿人”“你获得了纸灰气味”——用图标+粒子表现即可）。

------

## 六、测试标准

每段叙事内容完成后，需通过以下自检：

1. **主题关联**：这段内容是否推动玩家思考“自我来源”？
2. **文化融合**：湖湘元素是否自然融入机制或情感？
3. **玩家自主**：是否保留玩家解读空间，而非强加答案？
4. **情感真实**：NPC是否像“活过的人”，而非哲学传声筒？

------

> **最终目标**：
>  玩家通关后，不记得具体台词，但会默默问自己：
>  **“我的‘我’，是找到的，还是做成的？”**
>  ——这便是成功的叙事。

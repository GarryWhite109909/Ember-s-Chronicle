## 第一章：地窖 · 问器

> *主题：我是谁？——从“物”的身份开始怀疑*

### 🌑 幕1：苏醒

**场景描述**
 潮湿的岳麓书院废弃地窖。青砖墙渗水，角落堆满蒙尘的书箱、断砚、碎瓷。一缕月光从高窗斜入，照亮中央石台。

**玩家初始状态**

- 无名，无形，仅有一团微弱白雾。
- 屏幕中央浮现三件残物（静止悬浮）：
  - **青瓷片**（铜官窑釉色，裂纹如冰）
  - **湘竹简**（刻半句“道在器中”）
  - **苗银铃**（无声，内壁刻“守心”）

**交互**

- 玩家选择其中**两件**组合 → 形成初始外形（如：瓷+竹 = 半透明书卷状幽灵；竹+银 = 飘带形；瓷+银 = 碎片环绕）。
- 组合完成，雾气凝聚，一行古楷浮现：**“窑言”**。
- 同时，右下角出现**备注本图标**（线装册样式），提示：“可记录所见。”

> 💡 **设计意图**：首次“塑造自我”，名字与外形由玩家决定。

------

### 🕯️ 幕2：守院人

**NPC：老校工（70岁，驼背，手持油灯）**

- 他每日子时来地窖清扫，喃喃自语：“又漏雨了……她最怕潮。”

**目标**
 打开石台上的**乌木箱**（锁为三字转盘：生、辰、忌）。

**交互流程**

1. 玩家靠近 → 老校工打了个寒颤：“……有东西？”但看不见玩家。

2. 穿他

   （点击交互）→

   - 屏幕泛灰，浮现气味：**纸灰 + 艾草**

   - 读心文本（手写字体）：

     > “那年大火，我烧了她的日记……说好不哭的。
     >  可扫帚还是留着——她说，干净的地方，心才安。”

3. 解谜

   ：

   - 在场景找到**旧扫帚**（靠墙）；
   - 用“艾草”气味（自动触发）轻抚扫帚 → 老校工突然停步，从怀中掏出一张**泛黄纸片**，低声：“……是她的生辰。”

4. 玩家获得密码：**“壬午年五月初三”** → 转盘设为“生 / 辰 / 忌”。

> 💡 **机制教学**：气味可影响NPC情绪，推动剧情。

------

### ❓ 幕3：抉择

**乌木箱开启后**

- 内无实物，仅一本**空白线装册**，封面无字。
- 册页中央一行小楷：**“器无铭，何以传？”**

**系统提示**

> “可在此写下你的名字——但每字将消耗存在。”

**关键选择（二选一）**

- A. 写下“窑言”
  - 玩家手动输入（或确认预设名）；
  - 幽灵身形闪烁，获得气味：**窑烟**；
  - 备注本自动记下：“我名窑言。”
  - → **倾向【塑我】结局**
- B. 合上册子，不写
  - 册子化尘消散；
  - 获得气味：**纸灰**；
  - 备注本空白；
  - → **倾向【寻我】结局**

**章节结束**

- 地窖门吱呀打开，透出外面废墟的微光。
- 字幕淡入：**“第二章：废墟 · 观人”**





## 第二章：废墟 · 观人

> *主题：他人眼中的我，是否是我？*

### 🏚️ 幕4：渡口拾荒女

**场景描述**
 湘江边，文夕大火遗址。焦黑梁木斜插土中，瓦砾堆间长出野艾与桂花。黄昏，江风带腥。

**NPC：哑女（约20岁，衣衫补丁，背竹篓）**

- 她能模糊“看见”幽灵，常对空气低语，手指比划。

**目标**
 帮她找回埋在瓦砾下的**半块怀表**（表盖刻“1938.11.13”——文夕大火日）。

**交互流程**

1. 玩家靠近 → 哑女突然抬头，指向玩家位置，眼中含泪。

2. 穿她

    

   →

   - 气味：**桂香 + 铁锈**

   - 读心文本（稚嫩笔迹）：

     > “火好大……手好烫。
     >  阿爹把我推出窗，自己没出来。
     >  表停了，可我还记得他心跳。”

3. 解谜

   ：

   - 场景中有三处金属反光点；
   - 使用“铁锈”气味能力（需已收集≥1次）→ 高亮**真实怀表位置**（其余为弹壳、铜钱）；
   - 玩家引导哑女走向正确点 → 她跪地挖出怀表，紧紧贴在胸口。

4. 她抬头对玩家微笑，轻声（字幕）：“**谢谢……影子。**”

> 💡 **伏笔**：首次被称“影子”，玩家无反应（尚未知其意）。

------

### 📜 幕5：流浪学生

**场景延伸**
 哑女离开后，废墟深处亮起一盏煤油灯。一青年伏案疾书。

**NPC：青年（学生装，眼镜裂痕）**

- 自称在编写《长沙记忆录》，向路人征集故事。

**目标**
 获取他笔记中关于“地窖乌木箱”的真实记录。

**交互流程**

1. 玩家靠近 → 他自语：“若加点戏剧性，销量定好……”

2. 穿他

    

   →

   - 气味：**铜腥 + 纸灰**

   - 读心文本（印刷体，冰冷）：

     > “校工说烧了日记？假的。
     >  我亲眼见他藏进箱底——但写成‘焚毁’更悲壮。
     >  读者爱看牺牲，不爱看懦弱。”

3. 解谜

   ：

   - 笔记本摊开，关键页被墨涂黑；

   - 若玩家已解锁“茶烟”能力（第一章选A或此前收集≥3次茶烟）→ 凝视黑页，显出原文：

     > “箱中空无一物。校工骗自己，也骗我。”

   - 若无茶烟 → 无法读取，只能记录“他在说谎”于备注本。

4. 玩家获得真相：**乌木箱本就空**。

> 💡 **主题深化**：记忆可被篡改，“真实”依赖视角。

------

### ⚖️ 幕6：抉择

**青年合上笔记，望向江面**

- 他低声：“所有幽灵，都是活人不敢承认的自己。”

**系统提示**

> 哑女坐在不远处石阶上，轻轻摩挲怀表，似在等你。

**关键选择（二选一）**

- A. 显形安慰哑女
  - 玩家消耗存在值，短暂凝为可见形态（按初始外形）；
  - 哑女睁大眼，伸手触碰（穿过）→ 获得气味：**桂香**；
  - 她流泪微笑，将一朵干桂花放在地上 → 备注本自动记：“温柔值得被记住。”
  - → **倾向【塑我】结局**
- B. 隐身离开，继续追寻真相
  - 玩家直接走向渡口方向；
  - 哑女低头，怀表滑落 → 获得气味：**纸灰**；
  - 备注本无新增；
  - → **倾向【寻我】结局**

**章节结束**

- 江雾渐浓，远处乌篷船轮廓浮现。
- 字幕淡入：**“第三章：渡口 · 成我”**



## 第三章：渡口 · 成我

> *主题：当“我”不是物，也不是人，而是被抛弃的“影”——存在是否还有意义？*

### 🌙 幕7：摆渡人

**场景描述**
 深夜湘江渡口。木码头浸在月光与雾中，一艘旧乌篷船系于桩上，船头油灯将熄。远处岳麓山如墨剪影。

**NPC：老船夫（80岁，蓑衣斗笠，背对玩家）**

- 他手抚船板，低声哼湘剧《生死牌》：“……过江莫忘带影行。”

**目标**
 唤醒沉在船底的**记忆匣**。

**交互流程**

1. 玩家靠近 → 船夫不回头，只道：

   > “过江的，得带自己的影子。你有吗？”

2. 穿他

    

   →

   - 气味：**茶烟 + 窑烟**

   - 读心文本（苍老笔迹）：

     > “第一百零七次了……
     >  他们总把不要的东西留给我渡。
     >  上回是个学生，割下‘怕死’扔进江；
     >  上上回是位先生，切掉‘想哭’埋在橘子洲……
     >  你呢？你是谁不要的？”

3. 解谜

   ：

   - 使用“茶烟”能力凝视船板 → 显现**五道暗刻手印**（宫、商、角、徵、羽）；
   - 按**湘调韵律**敲击（节奏来自第一章校工扫地声或第二章哑女童谣）；
   - 正确后，船底升起**无锁木匣**，表面如镜。

> 💡 **机制回收**：前文声音线索成为解谜钥匙，强调记忆连贯性。

------

### 🪞 幕8：记忆匣（反转揭示）

**交互**
 玩家触碰木匣。

**视觉变化**

- 匣面如水，映出玩家当前外形（瓷/竹/银组合）；
- 镜像碎裂，化作**数十模糊人影**，背对站立，每人肩部缺一块——缺口形状，正是玩家外形。

**摆渡人缓缓转身，直视玩家**：

> “你不是窑里的器，也不是火里的灰。
>  你是**他们不敢带过江的那部分自己**——
>  怕、悔、爱、软、梦……
>  他们称你为‘影’，好让自己显得干净。”

**系统行为**

- **备注本自动翻开**，所有记录文字开始褪色；
- 屏幕中央浮现：**“你的记忆，是谁的碎片？”**

**关键选择（二选一）**

- A. 接受影子身份
  - 伸手触碰人影 → 匣中升起**七色气味结晶**（7种气味具象化）；
  - 进入幕9。
- B. 拒绝影子身份
  - 合上备注本，走向江心 → 直接触发【寻我】结局。

------

### 🌅 幕9：终局 · 影亦成形

**前提**：玩家选择A，面对七色结晶。

**交互**

- 悬停结晶，显示：
  - **窑烟** → 回归泥土（【寻我】）
  - **桂香** → 守护孩童（【塑我】）
  - **茶烟** → 散作无形（【超我】）
  - （其余气味导向混合，但归入三类）

**选择后**

- 玩家外形溶解，与所选气味融合；
- 摆渡人轻推船离岸，低语：**“这次，是你自己选的。”**





### 🏁 结局呈现

| 条件                              | 结局类型 | 画面                                     | 字幕               |
| --------------------------------- | -------- | ---------------------------------------- | ------------------ |
| **纸灰/铁锈 ≥4 且 未显形助人**    | 【寻我】 | 沉入江底，泥坯重塑，无名                 | “或许从未有我。”   |
| **桂香/艾草 ≥4 且 曾显形助人**    | 【塑我】 | 凝为新器（如桂花纹瓷瓶），陈列现代博物馆 | “我即我所为。”     |
| **茶烟/窑烟主导 且 接受影子身份** | 【超我】 | 化雾升空，次日孩童拾温热瓷片             | “名字，留给你取。” |

**最终画面**

- 黑屏，一行小字：**“这是第 __ 次苏醒。”**（数字 = 玩家收集的气味总数）
- 音效：湘江水声 + 一声遥远窑裂。

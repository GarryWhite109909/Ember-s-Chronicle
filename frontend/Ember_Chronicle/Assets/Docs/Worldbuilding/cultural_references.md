# 《窑言》湖湘文化元素清单

> 本清单严格遵循 **“可融入游戏机制、叙事或美术”** 原则，拒绝装饰性堆砌。
>  所有元素均服务于核心主题：**自我是寻找还是塑造？**

------

## 一、地理与空间

| 元素                     | 游戏应用               | 文化内涵                                                     |
| ------------------------ | ---------------------- | ------------------------------------------------------------ |
| **岳麓书院**             | 第一章主场景（地窖）   | “惟楚有材，于斯为盛”；理学重镇；“实事求是”思想发源地         |
| **文夕大火遗址**（1938） | 第二章主场景（废墟）   | 长沙抗战创伤记忆；文化浩劫；普通人如何在废墟中重建生活       |
| **湘江渡口**             | 第三章主场景           | “先天下之忧而忧”精神象征；生死过渡之地；湖湘人“敢为人先”的出发点 |
| **铜官窑遗址**           | 主角命名与气味系统来源 | 唐代釉下彩发源地；“器以载道”；窑火煅烧=自我成形              |
| **橘子洲**               | 背景提及（摆渡人台词） | 毛泽东《沁园春·长沙》；近代革命策源地                        |

## 四、语言与文本

| 元素                           | 游戏应用       | 文化内涵                                         |
| ------------------------------ | -------------- | ------------------------------------------------ |
| **“道在器中”**                 | 湘竹简刻字     | 王夫之哲学核心：真理不在空谈，而在具体之物       |
| **“器无铭，何以传？”**         | 第一章关键台词 | 化用《礼记》“铭者，自名也”；追问存在是否需被命名 |
| **湘剧五声音阶**（宫商角徵羽） | 第三章解谜节奏 | 湖南地方戏曲；旋律承载集体记忆                   |
| **长沙方言尾音**               | NPC台词细节    | 如“莫乱跑”“心才安”；增强地域真实感，不喧宾夺主   |
| **《楚辞》《船山遗书》**       | 场景书籍残页   | 屈贾之乡文脉；近代湖湘思想源头                   |

## 七、使用原则

1. **所有元素必须可交互或可感知**（如气味、声音、谜题），拒绝背景贴图式文化符号。
2. **不解释，只呈现**：玩家通过体验理解文化，而非阅读说明。
3. **服务于主题**：每个湖湘元素都需回答——它如何帮助探讨“自我是寻找还是塑造”？
4. **尊重历史创伤**：对文夕大火等事件，聚焦普通人的情感韧性，而非宏大叙事。

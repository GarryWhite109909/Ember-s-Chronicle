# 《窑言》主要角色档案

> 注：本作主角为非人存在，NPC均为短暂相遇的“过客”，但每人承载一种情感与湖湘记忆。
>  所有角色设计紧扣 **“自我是寻找还是塑造”** 主题，并服务于气味系统与解谜逻辑。

------

## 🌫️ 主角：窑言（You）

| 项目         | 内容                                                         |
| ------------ | ------------------------------------------------------------ |
| **身份**     | 被人类抛弃的“自我碎片”所凝聚的影子，附着于无主之物显形       |
| **外形**     | 由玩家在游戏开始时从三件残物中任选两件组合而成： • 青瓷片（铜官窑） • 湘竹简（岳麓书院） • 苗银铃（湘西） → 形态可为书卷状、飘带形、碎片环绕等 |
| **名字**     | 默认“窑言”，可于第一章选择是否正式命名                       |
| **性格**     | 无固定性格——其“人格”由玩家行为塑造： • 倾向共情 → 温柔、守护 • 倾向求真 → 冷静、疏离 • 接受虚无 → 宁静、超然 |
| **台词风格** | **无台词**。仅通过行动、外形变化、备注本内容表达“意志”       |
| **核心矛盾** | “我”是他人不要的部分，那“我”还能拥有自己的意义吗？           |
| **象征**     | 被放逐的潜能、未被承认的情感、湖湘文化中“克己”背后的牺牲     |

------

## 👴 第一章 NPC：老校工（守院人）

| 项目         | 内容                                                         |
| ------------ | ------------------------------------------------------------ |
| **姓名**     | 陈守仁（名字仅出现在乌木箱夹层纸片）                         |
| **年龄**     | 70岁                                                         |
| **背景**     | 岳麓书院最后一代守院人。1938年文夕大火中，为保护藏书冒死入火，却烧毁了亡妻手抄的《楚辞》。此后每日清扫地窖，假装她还在。 |
| **性格**     | 表面固执沉默，内心温柔自责；用“干净”掩盖“失去”。             |
| **台词风格** | 短句、重复、喃喃自语，带长沙方言尾音： • “又漏雨了……她最怕潮。” • “扫干净，心才安。” • “没主的魂，莫乱跑。” |
| **气味**     | **纸灰 + 艾草**（执念与守护交织）                            |
| **关键作用** | 引出“命名”抉择；揭示“记忆可被自我欺骗”。                     |
| **湖湘关联** | 体现湘人“重义守诺”与“隐忍担当”。                             |

------

## 🤫 第二章 NPC：哑女（拾荒者）

| 项目         | 内容                                                         |
| ------------ | ------------------------------------------------------------ |
| **姓名**     | 无名（村民称“哑妹”）                                         |
| **年龄**     | 约20岁                                                       |
| **背景**     | 文夕大火幸存孤儿。父亲将她推出火窗后葬身火海。她靠捡废墟旧物维生，坚信“东西有灵，不能丢”。能模糊看见幽灵，因她心中“影子”未被切割。 |
| **性格**     | 敏感、纯真、坚韧；用手语与世界沟通，眼神充满未说出口的话。   |
| **台词风格** | **无语音**，仅通过手语（字幕翻译）、表情、动作表达： • 指向玩家，眼中含泪 • 摸怀表，低头微笑 • 留下干桂花，轻轻挥手 |
| **气味**     | **桂香 + 铁锈**（温柔与创伤共生）                            |
| **关键作用** | 提供“显形 vs 隐身”道德抉择；首次称主角为“影子”。             |
| **湖湘关联** | 象征战火中普通人的韧性；桂花为长沙常见庭院植物，代表“微小却持久的温情”。 |

## 📖 第二章 NPC：流浪学生

| 项目         | 内容                                                         |
| ------------ | ------------------------------------------------------------ |
| **姓名**     | 周明远（笔记本扉页署名）                                     |
| **年龄**     | 25岁                                                         |
| **背景**     | 自称编写《长沙记忆录》，实为牟利篡改历史。曾采访老校工，却将“藏日记”写成“焚日记”，因“悲壮更卖座”。内心知道真相，但选择迎合大众想象。 |
| **性格**     | 聪明、功利、自我合理化；用“记录历史”掩饰“制造故事”。         |
| **台词风格** | 书面语混口语，带知识分子腔调： • “读者需要英雄，不是懦夫。” • “真实？真实太琐碎了。” • “所有幽灵，都是活人不敢承认的自己。” |
| **气味**     | **铜腥 + 纸灰**（欲望覆盖执念）                              |
| **关键作用** | 揭示“记忆可被权力重塑”；推动玩家质疑“寻找真我”的可行性。     |
| **湖湘关联** | 反讽近代“立言不朽”传统——当“言”为利所驱，道何存？             |

## ⛵ 第三章 NPC：摆渡人

| 项目         | 内容                                                         |
| ------------ | ------------------------------------------------------------ |
| **姓名**     | 无名（自称“渡影人”）                                         |
| **年龄**     | 外貌80岁，实际未知                                           |
| **背景**     | 百年湘江摆渡人，专渡“无主之物”与“不要之我”。他知晓所有被人类割舍的影子，并默默送它们过江。他是唯一能直视“窑言”本质的存在。 |
| **性格**     | 慈悲、通透、沉默如水；言语极少，但字字如钟。                 |
| **台词风格** | 古语韵律，短而深邃，带湘剧吟诵感： • “过江的，得带自己的影子。你有吗？” • “你是谁不要的？” • “这次，是你自己选的。” |
| **气味**     | **茶烟 + 窑烟**（释然与本源合一）                            |
| **关键作用** | 揭示终极反转；提供最终选择框架。                             |
| **湖湘关联** | 融合“先忧后乐”的渡世精神与“道在器中”的哲思；渡口为湖湘生死观的象征空间。 |

------

## 💎 设计原则总结

1. **每个NPC = 一种“自我处理方式”**
   - 校工：用仪式掩盖失去（压抑）
   - 哑女：接纳创伤并温柔活着（整合）
   - 学生：篡改记忆以求认同（扭曲）
   - 摆渡人：见证一切而不评判（超越）
2. **台词精炼，拒绝说教**
    所有哲学思想通过**行为、气味、环境**传递，而非长篇大论。
3. **湖湘元素内化为人格**
    不贴标签，而是让角色活在湖湘的历史肌理中。
4. **NPC不可重复互动**
    每人一生只被穿一次——强调**关系的唯一性与不可逆性**。

------

> 此档案可直接用于编剧、配音、美术设定。
>  所有角色服务于同一命题：**当你发现自己只是“影子”，你还会选择存在吗？**

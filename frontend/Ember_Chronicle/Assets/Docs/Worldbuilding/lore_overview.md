# 《窑言》世界观总纲

> **类型**：叙事解谜游戏
>  **主题**：自我是应该寻找，还是塑造？
>  **体量**：单人体验 · 约45–60分钟 · 三章九幕
>  **核心意象**：物品的幽灵、湖湘文脉、气味记忆、影子身份

------

## 一、核心设定

### 1. 主角：窑言

- **身份**：非人非鬼，而是**被人类抛弃的“自我碎片”所凝聚的影子**，附着于无主之物显形。
- **形态**：由玩家在游戏开始时从三件湖湘器物残片（青瓷片、湘竹简、苗银铃）中任选两件组合而成，外形即其初始“自我”。
- **名字**：“窑言”为默认名，象征**从窑火中诞生的言语**，亦谐音“遥言”（无人听见的低语）或“谣言”（不被承认的真实）。

### 2. 能力与限制

| 能力         | 说明                               | 隐喻                               |
| ------------ | ---------------------------------- | ---------------------------------- |
| **穿人读心** | 可穿过活人，读取其一段内心独白     | 试图从他人眼中寻找“我”             |
| **气味感知** | 每人散发独特“气味”，代表其情感本质 | 情感无法客观描述，只能主观体验     |
| **显形**     | 短暂凝为可见形态，消耗“存在值”     | 被看见=存在，但存在即消耗          |
| **备注记录** | 手动记录线索于线装册，模拟主动记忆 | 对抗遗忘，即是对“塑造自我”的实践   |
| **不可穿物** | 无法穿过墙壁、家具等非生物         | 幽灵被困于“关系”之中，无法脱离人类 |

### 3. 气味系统（7种基础）

| 气味     | 情感         | 湖湘关联               | 特殊能力（收集≥3次解锁） |
| -------- | ------------ | ---------------------- | ------------------------ |
| **窑烟** | 被遗忘的初心 | 铜官窑熄火余烬         | 回溯一件物品的最初用途   |
| **纸灰** | 未完成的执念 | 文夕大火、焚书         | 看见一段被烧毁的记忆残影 |
| **艾草** | 守护与牺牲   | 湘人端午挂艾、战地草药 | 安抚NPC情绪，促使其行动  |
| **铜腥** | 权力/欲望    | 湘军饷钱、商贾铜秤     | 揭示一段隐藏的谎言       |
| **茶烟** | 释然与放下   | 岳麓云雾茶、待客之道   | 看破一次表象，见真实     |
| **铁锈** | 创伤与固执   | 衡阳保卫战弹壳         | 定位金属物品             |
| **桂香** | 短暂的温柔   | 长沙老巷桂花糖         | 触发一段纯净童年记忆     |

> 气味收集倾向决定结局分支。

------

## 二、世界背景

### 1. 时空设定

- **时间**：模糊近现代（1930s–1980s），不指明具体年份，仅通过物品与记忆折射历史。

- 地点

  ：湖南长沙及周边，聚焦三个真实文化空间：

  - **岳麓书院地窖**（文脉封存之地）
  - **文夕大火遗址**（创伤记忆场）
  - **湘江渡口**（过渡与放下的象征）

### 2. 世界规则

- 此世界是**集体潜意识的边缘地带**，只有被强烈情感浸染过的物品才能在此显形。
- **人类看不见幽灵**，除非幽灵主动显形（代价高昂）。
- **时间非线性**：过去与现在的记忆碎片共存，玩家在其中穿行如翻阅一本残破日记。
- **存在依赖见证**：若长时间无人注意，幽灵将逐渐透明直至消散。

## 三、叙事结构：三章九幕

| 章节                    | 主题                             | 核心问题                           | 关键反转                                           |
| ----------------------- | -------------------------------- | ---------------------------------- | -------------------------------------------------- |
| **第一章：地窖 · 问器** | 我是谁？                         | 从“物”的身份开始怀疑               | 名字可写可不写——自我是否需命名？                   |
| **第二章：废墟 · 观人** | 他人眼中的我，是否是我？         | 记忆是否可靠？真相是否存在？       | 被称“影子”，但尚不知其意                           |
| **第三章：渡口 · 成我** | 若“我”本不存在，存在还有意义吗？ | 接受“影子”身份，是否等于放弃自我？ | **终极揭示：你不是物之灵，而是人割舍的“自我碎片”** |

> 每章3幕，共9幕，形成完整哲学弧光：**问 → 疑 → 决**。

## 四、核心哲学命题

### “自我是应该寻找还是塑造？”

- **寻找派**（本质主义）：相信存在一个“真我”，需回溯本源（如回归窑火、沉入江底）。

- **塑造派**（存在主义）：认为自我由选择与行动定义（如显形助人、立名传世）。

- 游戏立场

  ：

  > **“你本无我，故可成任何我。”**
  >  真正的自由，始于承认“自我”是流动的、关系的、可重构的——正如湖湘精神：**既重道统，又敢为天下先**。

### 影子的深意

- “影子”并非贬义，而是**被压抑的潜能**（荣格“阴影”理论）；
- 湖湘士人重“克己”“修身”，常切割情感以求“纯粹”，而你，正是那些被放逐的部分；
- **整合影子，人才完整**——这正是“塑造自我”的最高形式。

------

## 五、湖湘文化融入原则

- **不直接讲述历史**，只通过**物品、气味、方言词、民谣片段**折射；
- 所有场景、器物、台词均有真实文化依据（如铜官窑、文夕大火、岳麓书院“实事求是”匾）；
- 精神内核取自：
  - 屈贾之乡的忧患意识（“先忧后乐”）
  - 船山思想的“道在器中”
  - 近代湘人的“敢为人先”与“经世致用”

> 文化是土壤，不是装饰。

## 六、多结局系统

| 结局类型           | 触发条件                      | 核心意象             | 哲学回应                                  |
| ------------------ | ----------------------------- | -------------------- | ----------------------------------------- |
| **【寻我】· 归窑** | 纸灰/铁锈 ≥4 且 从未显形助人  | 沉入江底，泥坯重塑   | “或许从未有我。” → 接受虚无，即本真       |
| **【塑我】· 立言** | 桂香/艾草 ≥4 且 至少1次显形   | 凝为新器，陈列博物馆 | “我即我所为。” → 存在由行动定义           |
| **【超我】· 无名** | 茶烟/窑烟主导 且 接受影子身份 | 化雾升空，孩童拾瓷   | “名字，留给你取。” → 超越二元，存在即馈赠 |

- **终幕彩蛋**：黑屏显示“这是第 __ 次苏醒。”（数字 = 气味收集总数），暗示轮回与无限可能。

## 七、开发定位

- **目标平台**：PC / Mac（Steam / itch.io）
- **美术风格**：手绘水彩 + 半透明粒子，色调以青灰、赭石、墨黑为主，呼应湖湘山水与窑火
- **音效设计**：古琴 + 大筒（湖南胡琴） + 环境白噪音（滴水、江浪、风声）
- **无障碍**：气味用颜色+图标+文字三重标识；关键文本支持缩放
